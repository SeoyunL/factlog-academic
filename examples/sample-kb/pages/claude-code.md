# Claude Code

Claude Code is a command-line tool developed by Anthropic. It embeds an AI
assistant directly in the developer's terminal and supports a plugin system
for distributing reusable skills.

## Key facts

- Developed by: Anthropic
- Type: command-line tool
- Uses: Claude model family (Sonnet, Opus)
- Supports: plugin system
